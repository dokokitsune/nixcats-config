# NixCats config

This is my personal [`NixCats`](https://github.com/BirdeeHub/nixCats-nvim) Configuration with the Tokyonight color scheme. NixCats is Neovim Package Manager written in Nix that uses nixpkgs to download plugins. For lazy loading, it is recommended to use [`lze`](https://github.com/BirdeeHub/lze) or [`lz.n`](https://github.com/nvim-neorocks/lz.n).

## Included Dependencies

- ripgrep
- fd
- fzf
- wl-clipboard
- lazygit

## Auto Completion

- [Blink.cmp](https://github.com/Saghen/blink.cmp)
- [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig)
